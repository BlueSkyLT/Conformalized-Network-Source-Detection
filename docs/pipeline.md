# Pipeline Overview

This document describes the complete pipeline from raw network data to conformal prediction results.

## Pipeline Diagram

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                           PIPELINE OVERVIEW                                   │
└──────────────────────────────────────────────────────────────────────────────┘

 ┌─────────────────────┐
 │   Network Graph     │   SD-STGCN/dataset/<graph>/data/graph/<graph>.edgelist
 │   (.edgelist)       │
 └─────────┬───────────┘
           │
           ▼
 ┌─────────────────────┐
 │  Simulation Data    │   SD-STGCN/dataset/<graph>/data/SIR/<exp>_entire.pickle
 │  Generation         │   Generated by: run_entire_mix_random.sh
 │  (epidemic spread)  │
 └─────────┬───────────┘
           │
           ▼
 ┌─────────────────────┐
 │   SD-STGCN Model    │   SD-STGCN/output/models/<graph>/<train_exp>/
 │   Training          │   Run: main_SIR_mix_random.py or similar
 │   (TensorFlow)      │
 └─────────┬───────────┘
           │
           ▼
 ┌─────────────────────┐
 │   SD-STGCN Model    │   SD-STGCN/output/test_res/<graph>/<test_exp>/res.pickle
 │   Testing           │   Generated by: get_test_results() in utils/functions.py
 │   → res.pickle      │   Contains: predictions, ground_truth, inputs, logits
 └─────────┬───────────┘
           │
           ▼
 ┌─────────────────────┐
 │   Conformal         │   main.py
 │   Prediction        │   Reads res.pickle, computes conformity scores,
 │   (main.py)         │   calibrates thresholds, produces prediction sets
 └─────────┬───────────┘
           │
           ▼
 ┌─────────────────────┐
 │   Results           │   results/<graph>/<test_exp>/pow_expected<p>/
 │   - CSV tables      │   - coverage_table.csv
 │   - PDF plots       │   - set_size_table.csv
 │   - Pickle files    │   - coverage_boxplot.pdf, set_size_boxplot.pdf
 └─────────────────────┘
```

## Stage 1: Network Graph

**Input**: Network topology as an edgelist file.

**Location**: `SD-STGCN/dataset/<graph_name>/data/graph/<graph_name>.edgelist`

**Format**: Plain text file with one edge per line:
```
node1 node2
node1 node3
...
```

**Supported Graphs**:
- `highSchool` - High school contact network
- `bkFratB` - Fraternity network
- `sfhh` - SFHH conference network
- `ER` - Erdős-Rényi random graph
- `BA` - Barabási-Albert scale-free graph
- Others in `SD-STGCN/dataset/`

## Stage 2: Simulation Data Generation

**Purpose**: Generate epidemic simulation trajectories with known source nodes.

**Script**: `SD-STGCN/dataset/<graph>/code/run_entire_mix_random.sh`

**Usage**:
```bash
./run_entire_mix_random.sh <n_frames> <n_samples> <model> <min_src> <max_src> \
                           <min_R0> <max_R0> <min_gamma> <max_gamma> <instance>
```

**Parameters**:
| Parameter | Description |
|-----------|-------------|
| `n_frames` | Number of time frames to sample (typically 16) |
| `n_samples` | Total samples to generate (e.g., 21200 for training, 8000 for test) |
| `model` | Propagation model: `SIR` or `SI` |
| `min_src`, `max_src` | Range of source node counts |
| `min_R0`, `max_R0` | Range of basic reproduction number |
| `min_gamma`, `max_gamma` | Range of recovery rate |
| `instance` | Graph instance ID |

**Output**: `SD-STGCN/dataset/<graph>/data/SIR/<exp_name>_entire.pickle`

## Stage 3: SD-STGCN Model Training

**Purpose**: Train the graph neural network to predict source probabilities.

**Scripts**: `SD-STGCN/main_SIR_mix_random.py` and variants

**Input**:
- Training simulation data
- Graph structure

**Output**: `SD-STGCN/output/models/<graph>/<train_exp_name>/`
- TensorFlow checkpoint files
- Model metadata

## Stage 4: SD-STGCN Model Testing → `res.pickle`

**Purpose**: Generate per-node source probability predictions.

**Trigger**: Called automatically by `main.py` via `get_test_results()` if `res.pickle` doesn't exist.

**Function**: `utils/functions.py::get_test_results()`

**Input**:
- Test simulation data: `SD-STGCN/dataset/<graph>/data/SIR/<test_exp>_entire.pickle`
- Trained model: `SD-STGCN/output/models/<graph>/<train_exp>/`

**Output**: `SD-STGCN/output/test_res/<graph>/<test_exp>/res.pickle`

**Schema**:
```python
{
    'predictions': [batch1, batch2, ...],   # Each batch: (B x N x 2) probability array
    'ground_truth': [batch1, batch2, ...],  # Each batch: (B x N) one-hot source labels
    'inputs': [batch1, batch2, ...],        # Each batch: (B x N) initial states (0/1/2)
    'logits': [batch1, batch2, ...]         # Each batch: (B x N x 2) raw logits
}
```

## Stage 5: Conformal Prediction (`main.py`)

**Purpose**: Produce statistically-valid prediction sets with coverage guarantees.

**Input**:
- `res.pickle` from Stage 4
- Graph topology
- Configuration parameters (α levels, power threshold, methods)

**Process**:
1. Load and flatten batched predictions
2. Split into calibration (95%) and test (5%) sets
3. For each enabled method (`set_recall`, `set_prec`, `set_min`, etc.):
   - Compute conformity scores on calibration set
   - Determine threshold for each confidence level α
   - Construct prediction sets for test samples
   - Evaluate coverage and set size
4. Repeat for `mc_runs` Monte Carlo iterations
5. Generate summary statistics, tables, and plots

**Methods Available**:
| Method | Description | Score Function |
|--------|-------------|----------------|
| `set_recall` | Recall-based conformal score | `recall_score()` |
| `set_prec` | Precision-based conformal score | `avg_score()` |
| `set_min` | Minimum probability score | `min_score()` |
| `ADiT_DSI` | ADiT baseline (SI model only) | Monte Carlo sampling |
| `ArbiTree_CQC` | Tree-based structured prediction | Maximum spanning tree |

## Stage 6: Results Output

**Location**: `results/<graph>/<test_exp_name>/pow_expected<pow_value>/`

**Files**:

| File | Description |
|------|-------------|
| `coverage_table.csv` | Coverage rates: (method × α-level) |
| `set_size_table.csv` | Average set sizes: (method × α-level) |
| `time_cost_table.csv` | Runtime in seconds: (method × α-level) |
| `coverage_boxplot.pdf` | Visual comparison of coverage |
| `set_size_boxplot.pdf` | Visual comparison of set sizes |
| `calib_index_repeat<i>.npy` | Calibration indices for MC run i |
| `set_recall_repeat<i>.pickle` | Cached results for recall method |
| `set_prec_repeat<i>.pickle` | Cached results for precision method |
| `set_min_repeat<i>.pickle` | Cached results for min method |

## Path Conventions

### Experiment Naming

Training experiment names follow the pattern:
```
SIR_nsrc<min>-<max>_Rzero<min>-<max>_gamma<min>-<max>_ls<n_samples>_nf<n_frames>
```

Example: `SIR_nsrc1-15_Rzero1-15_gamma0.1-0.4_ls21200_nf16`

### Directory Nesting

```
<root>/
├── SD-STGCN/dataset/<graph>/data/graph/<graph>.edgelist
├── SD-STGCN/dataset/<graph>/data/SIR/<exp>_entire.pickle
├── SD-STGCN/output/models/<graph>/<train_exp>/
├── SD-STGCN/output/test_res/<graph>/<test_exp>/res.pickle
├── data/<graph>/graph/<graph>.edgelist           (cached copy)
├── data/<graph>/test_res/<test_exp>/res.pickle   (cached copy)
└── results/<graph>/<test_exp>/pow_expected<p>/
```

## Complete Example Workflow

```bash
# 1. Generate training data
cd SD-STGCN/dataset/highSchool/code
./run_entire_mix_random.sh 16 21200 SIR 1 15 1 15 0.1 0.4 0

# 2. Generate test data
./run_entire_mix_random.sh 16 8000 SIR 1 15 1 15 0.1 0.4 0

# 3. Train SD-STGCN model (from SD-STGCN directory)
cd ../../../
python main_SIR_mix_random.py  # Uses default parameters

# 4. Run conformal prediction
cd ../../../
python main.py \
    --graph highSchool \
    --train_exp_name SIR_nsrc1-15_Rzero1-15_gamma0.1-0.4_ls21200_nf16 \
    --test_exp_name SIR_nsrc1-15_Rzero1-15_gamma0.1-0.4_ls8000_nf16 \
    --pow_expected 0.5 \
    --prop_model SI \
    --set_recall 1 --set_prec 1 --set_min 1

# 5. Check results
ls results/highSchool/SIR_nsrc1-15_Rzero1-15_gamma0.1-0.4_ls8000_nf16/pow_expected0.5/
```
